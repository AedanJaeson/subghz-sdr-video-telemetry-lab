# Final Report Template

## Abstract

Briefly describe the project and final result.

## Motivation

Explain why an intentionally low-rate CC1101 image link is useful as a learning lab.

## Hardware

List verified hardware and relevant constraints.

## System Architecture

Show the full transmit, RF, receive, and decode chain.

## Packet Protocol

Describe packet fields, CRC, sequence tracking, and payload limits.

## RF Link

Document frequency, modulation settings, power assumptions, antennas, and legal/safety constraints.

## Receiver Design

Explain the GNU Radio and Python receiver chain.

## Results

Include screenshots, plots, packet-success tables, reconstructed images, and failure cases.

## Limitations

Be honest about throughput, reliability, latency, and legal constraints.

## Future Work

Discuss FEC, interleaving, better receiver synchronisation, better antennas, and cleaner viewer tooling.
